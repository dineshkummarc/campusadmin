# Introduction #
Welcome to  ECIT!

ECIT - Electronic Course Information Technology

ECIT has a suite of selective cores specifically tailored to the requirements of school industry. 
ECIT is designed considering range of management functions within school campus.
With the use of ECIT, staff can be more accountable as it helps to know the track performance of each department
in just few seconds. Almost all departments within school industry (e. g. admission,
administration, finance etc) can be optimized and granted.
ECIT supports to assign the responsibilities to employee staff and can reduce time wastage and can speed up the
administrative functions. Core functions like admissions, library management, transport management,
students activity in short entire range of college functions can be well performed by ECIT.

# System Requirements #
* A web server that can execute PHP
* A password-protected MySQL database server connection, and a database on which the user of the connection has full permissions rights (i.e. SELECT, DROP, CREATE and UPDATE)
* PHP 5.3 or later
* PHP must be run as the same system user that owns the directory where ECIT will be installed.

# Installation #
1. Upload ECIT to the web directory of your choice.
2. Create a new MySQL database for ECIT to use.
3. Browse to the ECIT folder and you will be redirected to the installation process.
4. Fill out the form, click install, and that's it!
5. You are now ready to use ECIT.

# Known Issues #

- On Windows Server WAMP / XAMPP  need to rewrite apache serveice : click on the WAMP/XAMPP icon and goto apache --> Apache Modules --> rewrite_modle  make sure this rewrite_module is chacked.
